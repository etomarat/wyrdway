## Contribute to the Code
If you have the capabilities to contribute to the code, your help is precious!  
Create [pull requests](https://github.com/nesbox/TIC-80/pulls) with the changes you want to make.

If you want ideas of what contributions to add, you can:
- look at the issues that are on the TODO list of the latest [project](https://github.com/nesbox/TIC-80/projects).  
- look at the issues [labelled with "bug"](https://github.com/nesbox/TIC-80/issues?q=is%3Aopen+is%3Aissue+label%3Abug).
- sort the issues by [number of likes](https://github.com/nesbox/TIC-80/issues?q=is%3Aissue+is%3Aopen+sort%3Areactions-%2B1-desc) to find out what the community is expecting the most.  
- the code editor still needs improvements, look at the [related issues](https://github.com/nesbox/TIC-80/issues?q=is%3Aopen+is%3Aissue+label%3A%22editor%3A+code%22).

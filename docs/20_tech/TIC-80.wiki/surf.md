`surf`

## Description
![surf](https://github.com/nesbox/TIC-80/assets/26139286/24052c42-d8e9-4a9d-97ef-1a95c023fd67)

Opens the carts browser to browse local carts and carts on the official tic80.com website from any platform.

Once a cart from the website is loaded, you can save it locally by using the `save` console command.
